This script fully removes SNAPS and apps from it, INCLUDING FIREFOX!
Make sure to run this script only ONCE, otherwise your current firefox **WILL BE PURGED**.

# Installation
1. Clone this repo by:
```bash
git clone https://github.com/beamyyl/better-ubuntu.git
```
2. Go into the cloned folder:
```bash
cd better-ubuntu/
```
3. Make the script executable:
```bash
chmod +x setup.sh
```
4. Run the script (will require sudo password)
```bash
./setup.sh```
